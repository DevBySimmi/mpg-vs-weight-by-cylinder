# 🚗 MPG vs Weight by Cylinder 📊

A Python data visualization project that explores the relationship between **car weight** and **miles per gallon (MPG)** for cars with different numbers of cylinders.

## 📊 Project Overview

This project creates a scatter plot to visualize how a car's weight affects its fuel efficiency.

The data is grouped into:

- 🔴 4-cylinder cars
- 🟢 6-cylinder cars
- 🔵 8-cylinder cars

Regression lines are added to show the general trend for each cylinder group.

## 📈 Graph

The graph compares:

- **X-axis:** Weight
- **Y-axis:** MPG
- **Groups:** 4, 6, and 8 cylinders

### Main Observation

The graph generally shows that as vehicle weight increases, **MPG tends to decrease**.

This means heavier cars generally have lower fuel efficiency.
